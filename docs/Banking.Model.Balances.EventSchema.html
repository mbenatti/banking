    banking vdev Banking.Model.Balances.EventSchema View Source

    The Balances of an bank account

    This schema control every fund added or removed of an account, the final balance is represented by balance field

    Each event have the previous event t/0 as a parent represented as parent

    The type is an Enum Banking.Model.Enums.BankOperationEnum.t/0 corresponding about Bank Account operation, the values accepted are "DEPOSIT", "TRANSFER_ISSUED", "TRANSFER_RECEIVED" and "WITHDRAWAL"

    The changeset for t/0

    Responsible also to set the sign of quantity_moved by Banking.Model.Enums.BankOperationEnum.t/0

    Do not accept the balance as params the balance is calculate by the changeset/2 using the previous t/0(:parent) using the balance of parent and the current quantity_moved
